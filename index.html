    Projects

    These are the some of the projects that I did during my career and undergraduate period. Codes of these projects are not available due to the copyright issues. All the other project details are available in my github account with codes.

    Unsupervised/self-supervised learning to spontaneously acquire infant-level perception

    This project is supervised by Dr. Muhammad Haris Khan at MBZUAI and Dr. Daniel Harari at Weizman Institute of Science, Israel.

    Using image analysis to estimate the density of blood cells- Omdena

    This project aims to estimate the density of various blood cell types based on visual data acquired by the HemoScope device which is developed by LiteBC (project partner). The videos/images will be analyzed and transformed into single numbers representing the quantities of various parameters: white blood cells count, red blood cells count, haemoglobin levels, etc. The training and testing processes will be based on both annotated data and results of traditional blood tests. Finally, blood count results will be given in 30 seconds non-invasively.

    Retina Multi Stages Formation/Deformation Detection- Omdena

    This project investigated the opportunity to explore the ML solution to detect the retina deformation and supporting the Diabetic Macular Edema (DME) identification. After preprocessing the images, we extracted the nerve fibre layer (above area of the retina) and coefficient of the itâ€™s curve using vertical projection since it is the most significant of detecting DME. Those coefficients were used to train different models and xgboost method gave the best results. Different experiments were conducted to find the best model. The performance of the solution was tested using the various healthy and DME data. I worked as a ML engineer and lead the model building task.

    Object Detection for Video Surveillance Applications

    At PromiseQ, I am working on developing state-of-the-art algorithms/models for object detection in CCTV applications. I am mainly focused on removing false alarms and improving the accuracy of the overall system.

    Using Satellite Imagery for Reforestation in Madagascar - Omdena

    I worked on developing a machine learning-based model using satellite images that will allow monitoring more effectively the land reforest. AI model(s) will be trained using remote sensing data and the models will be able to detect the trees and determine the biomass of land given an image.In the end, the model will provide an estimation of the number of individual trees in a particular area. I was the task lead of the model building and testing task in given time frame.

    Non invasive blood glucose measurement

    I developed a machine learning model to predict blood glucose level using non invasive method. Bio impedance, Skin Thickness, Melanin level and Heart Rate were taken as the physiological parameters and real blood glucose level was used as the reference value. Machine learning model was trained to obtain the bias values and constant value. Skin related parameters can be used to remove individual biases.

    Dropcatching system to catch competitive domains

    I was a key member (then the team leader) of a team that developed a state-of-the-art Domain Drop Catcher for a European Domain Registrar which uses a data-driven intelligent approach to identify and catch the most valuable domains. Distributed drop catcher consists of hundreds of concurrent bots that run at speeds less than a microsecond to achieve very high drop catching rates. We use the brute force method, delivering create commands on exact domain drop time, and AI-based domain drop time predicting methods depending on the registry to achieve a higher success rate with less than 50ms latency. The algorithm achieved superior real-time performance compared to the state-of-the-art algorithms

    Speeqo- Student Platform

    This is a project of developing a platform for students to submit their assignments and schoolwork. I extracted features from the audio signals and developed a model to give feedback to the teacher about the studentâ€™s emotional state

    Developing a hearing aid

    As for the final year project, we developed a prototype of a behind the ear hearing aid which enhances speech in noisy environments as the first phase of manufacturing hearing aids in Sri Lanka with the help of Wickramarachchi Institute of Speech and Hearing. I mainly worked in the Algorithm Development area where I built algorithms using MATLAB for Noise Reduction using Wavelet Transform, Feedback Reduction using Adaptive Filters and Frequency Shaping.

    Develop a wearable device for human stress detection

    In my internship at Synergen Technology Labs, I completed the first stage of wearable device to detect human stress using a wearable device. Algorithms were implemented to get readings from physiological signals such as GSR, Skin Temperature, Heart Rate, Body Acceleration and get a value for stress. Machine learning model was used to identify the type of the stress as Relax, Cognitive stress, Emotional Stress, and Physical Stress

    3D printed Electronic stethoscope

    Developed a Low-cost 3D printed electronic stethoscope which could detect heart sound, analyze through MATLAB software using ADAFRUIT feather board. Obtained signals were compared with the heart sound obtained from Biopac stethoscope. Analog Low pass filter which was implemented using op-amps was used to get only heart rate and remove other noises.

    The controlled environmental monitoring system

    Our team developed a system to monitor controlled system such as propagators. When humidity, Temperature, and light level exceed predefined limits, it will send a message to the user and an alarm sound will trigger, Atmega 328p is used as the microcontroller and Atmel studio is used as the coding platform.

    Voting Platform for District Rotaract Conference

    We developed a front end of the voting platform for District Rotaract Conference using Angular 10. HTML, CSS and JS were used to develop the platform.

    FPGA based processor design

    I completed a processor design that has the ability to downsample any given image using ALTERA DE2-115 FPGA development board. Verilog HDL and Quartus 16.1 were used for the project. Output and Input pictures were compared using python

    Wall Following Robot

    I was working on the design of a fully autonomous robot that can follow a given wall using a PIC microcontroller. Ultrasonic sensors were used for sensing the distance from the wall to the robot and PID algorithm was used to adjust the speed according to the distance.

    Analogue Lux Meter

    A prototype of a lux meter was developed to detect the intensity of the light. An Analog circuit was developed and Arduino is used to display the calculated intensity. Industrial Luxmeter was used to compare the performance.

    ECG Monitoring

    Developed an electronic circuit to detect ECG signals from hand. The circuit was designed to remove noise at the circuit level. ADAFRUIT feather The board was used to transmit signals to MATLAB software.

    FIR Digital Filter Design

    I implemented a finite impulse response digital filter using Kaiser Window method for a given signal. MATLAB software is used as the coding platform.
